# 🦋 What-If · 人生推演

> *如果当初做了不同的选择，人生会怎样？*

## 📖 项目简介

**What-If** 是一个基于 AI 的人生场景推演项目，灵感来源于漫威 *What If...?* 系列。

用户可以输入人生中某个关键节点的真实场景（人物、时间、事件），系统通过 AI 模拟推演出"如果当初做了另一个选择"后可能的结果，帮助用户：

- 🔄 **重温遗憾** — 高中没有表白的暗恋、没有勇气做的决定
- 💬 **弥补遗憾** — 对已故亲人说出当初说不出口的话
- 🌍 **体验他人** — 进入别人分享的人生场景，以旁观者或参与者视角感受
- 📜 **数字遗产** — 将人生经历构建为可被后人探访的"数字档案"

## ✨ 核心特性（规划中）

| 特性 | 描述 | 状态 |
|------|------|------|
| 场景构建 | 用户描述关键人物、时间、事件，AI 构建沉浸式场景 | 📝 构思中 |
| 选择推演 | 在关键节点做出不同选择，查看推演结果 | 📝 构思中 |
| 共享场景 | 匿名分享人生场景，让他人代入体验 | 📝 构思中 |
| 数字遗产 | 以第一人称整理记忆，构建可探访的人生档案 | 📝 构思中 |
| 多模态输入 | 支持文字、声音、影像等多种输入方式 | 📝 构思中 |
