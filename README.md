# 🦋 平行人生（What If）

> *用 AI 重现你的过去，让你活出另一种可能。*

## 📖 项目简介

**平行人生** 是一个基于 AI 的人生场景推演产品。用户可以回到人生中某个关键时刻，做出不同的选择，亲历另一条人生轨迹。

与直接和 AI 聊天不同，平行人生提供：
- 🧠 **持久记忆** — 人物建档后性格稳定，不会"失忆"
- 📖 **叙事引擎** — 推演结果以精心排版的故事呈现，有节奏感和情感弧线
- 💾 **完整存档** — 每段平行人生可命名、存档、回看
- 🌍 **社区共鸣** — 匿名分享遗憾，他人可续写你的平行人生

## 🎬 核心使用场景

| 场景 | 示例 | 情感浓度 |
|------|------|----------|
| 青春遗憾 | 当年没敢表白 | ⭐⭐⭐ |
| 职业选择 | 当初是否该辞职创业 | ⭐⭐⭐ |
| 学业节点 | 如果认真准备高考 | ⭐⭐⭐ |
| 关系修复 | 和已经断联的朋友和解 | ⭐⭐⭐⭐ |
| 家庭遗憾 | 对去世长辈未说出口的话 | ⭐⭐⭐⭐⭐ |

## 🧩 核心功能模块

### 🌍 世界构建（World Builder）
用户在推演前搭建"那个时代的世界"：人物库、事件节点、时代背景。

### ⚙️ 推演引擎（Simulation Engine）
AI 以"那个时代的你"为视角推演，关键人物以建档的性格、口吻发言，产出故事级叙事文本。

### 📂 存档与回看（Archive）
每次推演形成完整"平行人生"档案，可命名、打标签、分享。

### 💬 社区共鸣（Community）
匿名发布遗憾节点，其他用户续写、点赞、评论："我也有过一样的遗憾"。

## 🗺️ 开发路线图

- **阶段一 MVP（4–6 周）** — 简单事件填写 + AI 生成平行结局 + 基础存档
- **阶段二 核心产品（1–3 个月）** — 完整人物库 + 对话式推演 + 叙事引擎
- **阶段三 差异化壁垒（3–6 个月）** — 语音克隆 + 社区 + 心理健康边界
- **阶段四 终极愿景** — 多模态沉浸 + 空间计算交互
